TC-4: 企业能源成本概况-选择不同时间周期

Set up

本地部署应用
https://github.com/yoitech/steel-energy-management-app

Actions

1.登录进入智能能效管理系统。
2.进入首页-能源价值-能源成本计算-企业能源成本概况。
3.选择2018年1月1日至2018年12月31日，点击查询按钮，查看日，月，季度，年能源成本的数据。
4.选择2019年1月1日至2019年1月20日，点击查询按钮，查看日，月，季度，年能源成本的数据。

Expected Results

1.成功登录。
2.跳转成功至企业能源成本概况。
3.数据成功展示。
4.数据成功展示并和步骤3的数据有差异。

Breakdown
